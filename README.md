# 📚 About This Repository

This repository contains **typed coding notes for multiple programming languages**.

Unlike traditional notes, these are **execution-based learning notes** where each section contains:

* Practical code
* Concept explanation
* Anchor-based navigation
* Runnable mini examples

These notes are designed to help developers **learn programming by running code instead of memorizing theory.**

---

# 🧠 Learning Philosophy

Instead of reading theory, you will:

1️⃣ Run a code snippet
2️⃣ Observe the output
3️⃣ Understand the concept
4️⃣ Move to the next anchor section

Each concept is structured as **small runnable sections** to make learning interactive.

# 🔌 Recommended VS Code Extensions

These extensions improve the learning experience.

---

## 📌 Comment Anchors

Extension: **Comment Anchors**

Used to mark sections in code.

Example:

```
# ANCHOR: Variables
```

```
# ANCHOR: Loops
```

This allows quick navigation across concepts inside VS Code.

---

## ⚡ Code Runner

Extension: **Code Runner**

Run code instantly without creating complex setups.

Shortcut:

```
Ctrl + Alt + N
```

---

## 🟨 JavaScript Runner

Extension:

```
JavaScript Code Runner
```

Useful for executing JavaScript snippets quickly.

---

# 🧪 How To Use These Notes

1. Open a language folder

Example

```
python/
javascript/
```

2. Find an anchor section

```
# ANCHOR: Functions
```

3. Run the code snippet.

4. Observe the output.

5. Understand the concept explained in comments.

6. Continue to the next anchor.

This makes the repository behave like a **self-paced coding lab.**

Create a Pull Request.
